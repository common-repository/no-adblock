== Description ==

This plugin will detect adblock users and blocking them from seeing your site content. 

= Why No Adblock ? =

Nowadays, millions of sites survive to bring out valuable information for free to its visitors. This is achieved through having advertisements on their site to finance its existence.

== Installation ==

Use WordPress' Add New Plugin feature, searching "No Adblock", or download the archive and:

1. Upload `noadblock` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Adblock located in Left Sidebar and customize behavior as needed

== Frequently Asked Questions ==

= The plugin doesn't work, why? =

Try de-activate and activate it again
